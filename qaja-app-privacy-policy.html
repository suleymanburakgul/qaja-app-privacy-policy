Privacy Policy – QAJA

Teos Energy GmbH operates QAJA in compliance with GDPR (EU General Data Protection Regulation).

Data Collection

We collect only essential data required to operate the platform:

  • Account registration information (name, email)
  • Project and task details voluntarily added by users
  • Technical information such as device type and basic usage logs

We do not use Google Analytics or third-party tracking tools.

Data Usage

Data is used only for:

  • Providing platform functionality
  • Improving user experience
  • Securing the system and preventing misuse

We do not sell or share user data with third parties.

Data Storage

User data is stored securely on EU servers with encrypted transmission. Users can request data deletion at any time by contacting us.

User Rights (GDPR)

Users have the right to:

  • Access their data
  • Correct or delete data
  • Object to data processing
  • Request data portability
